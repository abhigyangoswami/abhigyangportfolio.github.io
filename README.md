# Abhigyan's Personal Portfolio
This repository contains the source code for my professional portfolio website, hosted via GitHub Pages. The site is designed to showcase the intersection of my logic-driven hobbies (Chess) and my technical skills (Programming & Robotics).

**[🔗 Explore the Live Site](https://abhigyangoswamiportfolio.github.io)**

## 🚀 Site Features
- **Custom Glow UI:** A high-tech aesthetic inspired by modern IDEs and dark-mode themes.
- **Responsive Grid:** A two-column project showcase designed for both desktop and mobile.
- **Interactive Elements:** Floating STEM-themed icons with parallax effects and smooth-scroll navigation.

## 🛠️ Built With
- **HTML5:** For structure and foundation.
- **CSS3:** Custom animations, Flexbox/Grid layouts, and "Glassmorphism" effects.
- **JavaScript:** Logic for scroll-based animations and back-to-top functionality.

## 📂 Featured Projects in the Lab
- **Wordle for Chess:** A Python-based CLI strategy game.
- **Arduino PWM Simulation:** Hardware intensity control with C++.
- **Secure Password Generator:** Cryptographically secure utility.
